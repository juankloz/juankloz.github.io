# Sitio personal JuanKLoz

Sitio estático preparado para publicarse gratuitamente en GitHub Pages con la cuenta `juankloz`.

## Dirección pública

Al publicarlo como sitio personal quedará en:

`https://juankloz.github.io/`

## Publicación desde el navegador

1. Inicia sesión en GitHub con la cuenta `juankloz`.
2. Crea un repositorio **público** con el nombre exacto `juankloz.github.io`.
3. En el repositorio, selecciona **Add file → Upload files**.
4. Sube el contenido de esta carpeta a la raíz. `index.html` debe quedar directamente en la raíz, no dentro de otra carpeta.
5. Confirma la carga con **Commit changes**.
6. Abre **Settings → Pages**.
7. En **Build and deployment**, elige **Deploy from a branch**.
8. Selecciona la rama `main`, la carpeta `/(root)` y pulsa **Save**.
9. Revisa el estado de publicación en **Settings → Pages** o en la pestaña **Actions**.

## Archivos principales

- `index.html`: contenido, metadatos y enlaces.
- `styles.css`: diseño adaptable a celular y computador.
- `script.js`: menú y animaciones.
- `assets/og-image.png`: imagen para compartir en redes.
- `assets/favicon.svg`: icono del navegador.
- `.nojekyll`: indica a GitHub Pages que publique los archivos tal como están.

## Actualizaciones futuras

Para cambiar textos o enlaces, edita `index.html` desde GitHub y confirma con **Commit changes**. GitHub Pages volverá a desplegar el sitio automáticamente.
